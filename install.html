Install Hidaya

Mobile (iOS & Android)

  1. Open Hidaya in Safari (iOS) or Chrome (Android).
  2. Tap the Share button (iOS) or Menu (Android).
  3. Select Add to Home Screen.

Windows, Mac & Android

Install Hidaya as a native app for the best experience.

Automatic install unavailable

  1. Open browser menu (⋮ or ⋯).
  2. Select Install Hidaya or Add to Home Screen.
  3. Or look for the install icon in the address bar.
